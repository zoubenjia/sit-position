# 低头族检测 — 手机端坐姿监控

利用手机传感器检测低头姿势，AI 面部分析检测疲劳状态，及时提醒用户注意颈椎健康。

## 功能

### 标准模式 — 低头检测
- **实时角度仪表盘** — 彩色圆盘显示手机倾斜角度（0-90°）
- **三级状态判定** — 姿势良好（绿）→ 注意姿势（黄）→ 低头警告（红）
- **智能提醒** — 低头超时震动 + 声音提醒，防抖机制避免误报
- **躺姿识别** — 自动检测躺下姿势，暂停误报提醒
- **自定义阈值** — 角度阈值、提醒延迟、冷却时间均可调

### 精准模式 — AI 疲劳检测
- **眨眼频率** — 通过 EAR（眼睛纵横比）实时追踪眨眼次数
- **哈欠检测** — 通过 MAR（嘴巴纵横比）统计 5 分钟内哈欠次数
- **疲劳分级** — 正常 / 疲劳 / 严重疲劳三级预警
- **闭眼检测** — 连续闭眼超过 2 秒触发严重疲劳警告

### 使用统计
- **今日概览** — 提醒次数、坏姿势时长、监控时长、坏姿势占比
- **周趋势图** — 7 天柱状图展示每日坏姿势分钟数
- **每日明细** — 按天查看详细数据，支持下拉刷新

## 原理

**标准模式**：通过加速度计判断手机倾斜角度：
- 手机竖直（0-50°）→ 用户平视，姿势良好
- 手机倾斜较大（50-70°）→ 用户微低头，注意提醒
- 手机接近水平（70-90°）→ 用户严重低头，触发警告

**精准模式**：通过前置摄像头 + Google ML Kit 面部检测：
- 计算眼睛纵横比（EAR）判断眨眼和闭眼
- 计算嘴巴纵横比（MAR）判断哈欠
- 综合分析眨眼频率和哈欠次数评估疲劳等级

## 快速开始

### 1. 安装 Flutter

```bash
# macOS
brew install flutter

# 或从官网下载: https://docs.flutter.dev/get-started/install
```

### 2. 初始化项目

```bash
cd sit_position_mobile
bash setup.sh
```

### 3. 运行

```bash
# 连接真机或启动模拟器
flutter run
```

> **注意**: 传感器功能需在真机上测试，模拟器无加速度计数据。精准模式需要前置摄像头。

## 可调参数

| 参数 | 范围 | 默认值 | 说明 |
|------|------|--------|------|
| 角度阈值 | 20-80° | 50° | 触发低头检测的倾斜角度 |
| 提醒延迟 | 5-60s | 15s | 低头多久后首次提醒 |
| 提醒冷却 | 10-120s | 30s | 两次提醒的最小间隔 |
| 良好判定次数 | 1-10 | 3 | 连续几次好姿势后恢复正常 |
| 采样频率 | 100-500ms | 200ms | 传感器读取间隔 |
| 震动提醒 | 开/关 | 开 | 触觉反馈 |
| 声音提醒 | 开/关 | 开 | 播放提示音 |

## 技术栈

- Flutter 3.x + Dart
- sensors_plus — 加速度计
- camera + google_mlkit_face_detection — 前置摄像头 AI 面部检测
- flutter_local_notifications — 本地通知
- vibration — 震动反馈
- audioplayers — 声音提醒
- shared_preferences — 设置与数据持久化
- intl — 日期格式化

## 自定义提示音

将 MP3 文件放在 `assets/sounds/alert.mp3` 即可自定义低头提示音。

## 隐私声明

- 所有数据处理在**本地设备**上完成
- 摄像头画面**不保存、不录制、不上传**
- 精准模式仅提取面部特征点数值，**不存储任何图像**
- 不收集任何个人信息
