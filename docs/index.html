Sit Monitor

用摄像头实时检测坐姿，智能图标指示具体问题，语音提醒纠正。开源、隐私优先。

🧑‍💻 姿势良好
😥 颈部前倾
🚨 多项问题
下载 DMG (macOS) GitHub
🎯

四项姿势检测

肩膀倾斜、头部前倾、头部侧倾、躯干前倾，实时角度计算，坐姿站姿通用。

🟢

动态图标指示

托盘图标实时变化：绿色=良好，橙色=单项问题，红色=多项问题，一眼看出哪里不对。

🔊

智能提醒

坏姿势持续 30 秒弹窗提醒，纠正后语音鼓励。久坐 45 分钟提醒活动。

🏋️

俯卧撑训练

语音引导就位，实时计数，姿势纠正，训练记录。还能和朋友对战 PK。

⏸️

离开自动暂停

人离开摄像头自动暂停浏览器视频，回来自动恢复。支持主流浏览器。

🌏

中英文双语

菜单、通知、语音播报全面本地化，一键切换。macOS 和 Windows 双平台。

🔒 隐私优先

所有图像处理在本地完成，摄像头画面不保存、不录制、不上传。程序只计算关节角度，不存储任何图像。代码完全开源。

快速开始

DMG 安装
下载 SitMonitor.dmg
Homebrew
brew install zoubenjia/tap/sit-monitor
pip
pip install git+https://github.com/zoubenjia/sit-position.git
